História de Usuário: Como Pesquisador quero me cadastrar para poder acessar o sistema

Cenário de Aceite

Dado que eu abra a página de cadastro;
Uma tela sera aberta solicitando todos os dados do pesquisador: Nome, email, CPF, RG, senha;
Quando eu inserir os dados solicitados;
Então o sistema enviará minha solicitação de cadastro ao administrador;
Quando chegar a solicitação, o administrador receberá uma notificação para validar o cadastro;
A partir do aceite do administrador, o pesquisador pode utilizar o sistema
