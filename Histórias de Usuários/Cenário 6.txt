História de Caso: Como administrador quero liberar o acesso que cada perfil cadastrado terá permissão de acessar em sua conta

Cenário de Aceite

Dado que eu abra o sistema
E ir na parte de novos usuários
E avaliar os dados de cada um
A partir do momento de que os dados do usuário são verdadeiros
O sistema enviará um código para os usuários confirmados
Quando chegar esse código para os usuários, o mesmo confirmaria o código no sistema e assim teria permissão para acessar sua conta.
